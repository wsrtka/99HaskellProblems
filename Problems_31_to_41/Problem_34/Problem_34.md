# Problem_34
### Difficulty: medium
Calculate Euler's totient function phi(m).
Euler's so-called totient function phi(m) is defined as the number of positive integers r (1 <= r < m) that are coprime to m.

Example:

```

* (totient-phi 10)
4
```

```
Î»> totient 10
4
```
