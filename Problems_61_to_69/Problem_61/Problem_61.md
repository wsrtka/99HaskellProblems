# Problem 61
### Difficulty: 
As defined in problem 54A.
An example tree:
Count the leaves of a binary tree
A leaf is a node with no successors. Write a predicate count_leaves/2 to count them.

Example:

```

% count_leaves(T,N) :- the binary tree T has N leaves
```
Example in Haskell:

```
λ> countLeaves tree4
2
```
